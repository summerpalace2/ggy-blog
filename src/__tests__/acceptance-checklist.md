# 飞书云文档块级交互 — 手工验收清单

> 使用方式：逐项操作，勾选通过项。

---

## 一、悬停控件验收

- [ ] **T1.1** 打开 /write 页面，将鼠标移动到任意标题块（H1/H2/H3）左侧 gutter 区域 → 拖拽把手和类型标签出现
- [ ] **T1.2** 将鼠标从标题块左侧移向标题文本内容区，再轻微移回左侧 → 控件不连续闪烁（200ms 缓冲）
- [ ] **T1.3** 对正文块（段落）重复 T1.1 和 T1.2 → 行为一致
- [ ] **T1.4** 鼠标从页面空白处（块之间的空隙）掠过 → 不触发右侧邻近块的控件
- [ ] **T1.5** 开始在一个块中输入文字（光标闪烁中）→ 移动到该块左侧时 hover 控件不出现（输入优先）

## 二、`/` 快捷插入菜单验收

- [ ] **T2.1** 在一个空正文块中输入 `/` → 斜杠菜单出现在光标下方
- [ ] **T2.2** 输入 `/` 后，左侧 hover 控件被压制（不显示）
- [ ] **T2.3** 用键盘上下箭头选择菜单项 → 高亮跟随移动
- [ ] **T2.4** 按回车确认 → 选中项插入成功（块类型改变）
- [ ] **T2.5** 按 Esc → 菜单关闭
- [ ] **T2.6** 在搜索框中输入"标题" → 菜单过滤显示相关项
- [ ] **T2.7** 在阅读态（read）下输入 `/` → 菜单不打开

---

## 三、评论锚点与正文回跳验收

- [ ] **T3.1** 选中一段文字，点击右侧 💬 按钮 → 评论输入框出现
- [ ] **T3.2** 输入评论内容并提交 → 正文左侧出现蓝色高亮竖线
- [ ] **T3.3** 无选区的块，点击 💬 按钮 → 块级评论创建成功
- [ ] **T3.4** 打开评论面板（顶部工具栏 💬 按钮）→ 刚创建的评论可见
- [ ] **T3.5** 在评论面板中点击某条评论 → 页面平滑滚动到对应块，触发闪烁动画
- [ ] **T3.6** 评论高亮（蓝色竖线）与普通选中高亮明显不同
- [ ] **T3.7** 块评论数 badge 显示正确（蓝色圆点上的数字）
