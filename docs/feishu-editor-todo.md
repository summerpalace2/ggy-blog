1. 先把命令系统和块插入闭环做稳
2. 再把目录、图片、代码块等常用能力补齐
3. 再做表格、公式、折叠、callout
4. 最后拆结构化数据和协作能力

## 7. 完成定义

当以下条件全部满足时，可认为第一阶段完成：
- 用户能在写作页完成文章标题和正文写作
- 能通过 `+` 和 `/` 插入主要内容块
- 能保存草稿和发布文章
- 能用键盘完成大多数结构编辑
- 页面刷新后草稿可恢复
- 页面整体稳定，无明显卡顿或丢失内容问题
